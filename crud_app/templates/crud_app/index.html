A little about the project

The aim is to create an API backend running on django framework that have endpoints to handle a basic school information system which can:

  • Create Students’ resource

  • Read Students’ resource

  • update and delete students.


By consuming the endpoints from the backend, I built a frontend page that handles

  • Creating Students’ resource

  • Listing Students’ resource

  • Showing Students’ resource

  • editing and deleting students on backend.
